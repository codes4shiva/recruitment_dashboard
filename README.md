# RecruitIQ вЂ” AI Recruitment Analytics Dashboard

RecruitIQ is a modern AI-powered recruitment marketing analytics dashboard built to help hiring teams track campaign performance, optimize hiring spend, and generate recruitment insights through a premium SaaS-style interface.

The platform provides real-time analytics, campaign management, performance visualization, and reporting tools for recruitment marketing campaigns across multiple hiring channels.

---

# Features

## Dashboard Overview

* Real-time recruitment campaign analytics
* KPI tracking cards
* Performance trend visualization
* Recruitment spend analytics
* Interactive charts and graphs

## Campaign Management

* Create recruitment campaigns dynamically
* Track campaign performance
* Monitor campaign spend and applications
* Multi-channel recruitment tracking

## Analytics & Insights

* Performance trend analysis
* Channel-wise spend distribution
* Conversion analytics
* Recruitment funnel tracking

## Reports System

* Generate performance reports
* Campaign analysis reports
* Channel analysis exports
* Cost analysis reports
* Conversion reports
* Downloadable analytics summaries

## Modern SaaS UI

* Premium dark-themed dashboard
* Fully responsive layout
* Interactive charts
* Smooth hover animations
* Enterprise-style analytics experience

# Installation

Clone the repository:

```bash
git clone https://github.com/codes4shiva/recruitment_dashboard.git
```

Move into the project directory:

```bash
cd recruitment_dashboard
```

Install dependencies:

```bash
npm install
```

---

# Run Development Server

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
